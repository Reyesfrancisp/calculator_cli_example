# Interactive Calculator CLI

A command-line interface (CLI) calculator written in Python. This application allows users to perform basic arithmetic operations (addition, subtraction, multiplication, division) through an interactive shell.

## Features
- **REPL Interface**: Continuous interaction loop.
- **Error Handling**: Gracefully handles division by zero and invalid inputs.
- **100% Test Coverage**: Verified by GitHub Actions.
